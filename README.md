## Overview

Electronic Queue Management System is a modern interface designed to improve customer experience inside electronic component stores by providing intuitive navigation, live stock visibility, nearby store integration and simplified product discovery.

✔ Modern Dashboard

✔ Product Categories

✔ Smart Search

✔ Store Availability

✔ Google Maps Integration

✔ Shipment Tracking
